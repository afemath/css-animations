• CSS3 Compound Transforms

  • Origin

    translate(x,y), translateX(x), translateY(y), translateZ(z), translate3d(x,y,z)

    Allows you to change the position on transformed elements.

  • Translate

    translate(x,y), translateX(x), translateY(y), translateZ(z), translate3d(x,y,z)

    Defines a translation, moving the element along the X, Y and/or Z axis.

  • Scale

    scale(x,y), scaleX(x), scaleY(y), scaleZ(z), scale3d(x,y,z)

    Defines a scale transformation along the X, Y and/or Z axis.

  • Skew

    skew(x,y), skewX(x), skewY(y)

    Defines a skew transformation along the X and/or Y axis.

  • Rotate

    rotate(angle), rotateX(angle), rotateY(angle), rotateZ(angle), rotate3d(x,y,z,angle)

    Defines a 2D rotation, the angle is specified in the parameter

  • Perspective

    perspective(n)

    Defines a perspective view for a 3D transformed element

  • Matrix

    matrix(n,n,n,n,n,n), matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

    Defines a 2D/3D transformation, using a matrix of six values for 2D and 16 values for 3D

  • Transitions

    [property] [duration] [delay]

    Transitions allows you to change property values smoothly (from one value to another), over a given duration.
